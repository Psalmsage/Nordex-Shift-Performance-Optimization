# Nordex-Shift-Performance-Optimization
# Overview
This project focuses on improving production efficiency at NorDex Manufacturing AS by analyzing the performance of different production shifts using data analytics and machine learning.


The company experiences productivity differences between morning, evening, and night shifts, affecting output, downtime, and operational costs. This project aims to identify the causes of these inconsistencies and provide data-driven solutions for optimizing shift performance.

# Objectives
- Analyze production shift performance
- Identify factors affecting productivity
- Reduce downtime and defect rates
- Improve workforce and machine allocation
- Build predictive models for operational optimization

# Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn


# Expected Outcomes
- Improved production consistency
- Reduced operational costs
- Better resource allocation
- Increased throughput and efficiency
- Real-time performance insights
